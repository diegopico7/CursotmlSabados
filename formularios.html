Formularios

Para insertar formularios se usa la etiqueta form y es de apertura y cerrado, para insertar un campo se usa la etiqueta input y es solo de apertura. A la etiqueta input podemos manipularla con los atributos correspondientes, para darle una descripcion o nombre al campo se usa la etiqueta label. Algunos atributos para los input importantes es placeholder, size, required, minlength, maxlength
